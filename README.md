# Library Management - Beginner React Vite Project

This is a beginner-friendly React project for teaching:

- State
- Events
- Conditional rendering
- List rendering
- React forms
- Controlled components
- Basic form validation
- `useState`
- `useEffect`
- Component lifecycle thinking
- React Router
- Layout routes
- Nested pages
- Protected route concept
- Add, edit, delete book CRUD

## Login Credentials

```txt
username: admin
password: admin123
```

## Run the Project

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually:

```txt
http://localhost:5173
```

## Important Note

This project uses hardcoded authentication and localStorage only for learning React. In a real MERN app, authentication and books data should come from backend APIs using Express, MongoDB, and secure authentication.
